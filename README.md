A Three.js solar system project built with Vite. It renders the Sun, planets, orbital motion, clickable planets, and planetary rings for Saturn and Uranus.

- 3D solar system built with Three.js
- OrbitControls for camera movement and zoom
- Textured Sun and planets
- Planet orbit animation
- Click a planet to log its name in the console
- Rings for Saturn and Uranus
- Starfield background


- index.html - page structure and overlays
- script.js - Three.js scene, animation, and interactions
- vite.config.js - Vite config for local dev and GitHub Pages
- img/ - texture assets
- docs/ - generated production build for GitHub Pages

- The project uses local texture imports from `img/`.
- If the page is blank on GitHub Pages, make sure the `docs/` folder is committed and Pages is pointed at `/docs`.
